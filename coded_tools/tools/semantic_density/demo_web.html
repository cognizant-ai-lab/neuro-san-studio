The Algorithm: 4-Step Pipeline

Extracted into ConfidenceAssessmentEngine

1. Diverse Beam Search
→
2. Token Probabilities
→
3. NLI Distance
→
4. Confidence Score

Generate multiple diverse answers → measure token-level probabilities → score pairwise contradiction via NLI → combine into a single 0–1 confidence score

Wrapping as a CodedTool

ConfidenceAssessmentTool — async invoke for agent networks

  1. 01
    CodedTool Subclass
    Implements async_invoke() — the standard interface for neuro-san tools
  2. 02
    Thread Pool for GPU
    asyncio.to_thread(engine.evaluate, question) — GPU inference blocks, so we offload to keep the event loop responsive
  3. 03
    Classmethod Singleton
    ConfidenceAssessmentEngine.get_instance() — load 7B + NLI models once, reuse across all requests
